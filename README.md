# Spotify × GenAI Prototype — Open Project 2026

## How to Deploy to Vercel

### Method 1: Drag & Drop (Fastest)
1. Go to [vercel.com](https://vercel.com) and log in (free account works)
2. Click **"Add New → Project"**
3. Drag the entire `spotify-genai-prototype` folder onto the Vercel dashboard
4. Click **Deploy** — your URL is live in ~30 seconds ✅

### Method 2: Vercel CLI
```bash
npm install -g vercel
cd spotify-genai-prototype
vercel --prod
```

### Method 3: GitHub
1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → New Project → Import from GitHub
3. Select the repo → Deploy
